## Overview

An interactive data visualization dashboard for exploring and analyzing datasets with beautiful charts, graphs, and real-time analytics.

## Features

- **Interactive Charts** - Zoom, pan, and filter visualizations
- **Multiple Chart Types** - Bar, line, pie, scatter, heatmap, treemap
- **Real-time Data** - Live data streaming and updates
- **Export Options** - Download charts as PNG, SVG, or PDF
- **Responsive Design** - Works on desktop and mobile
- **Custom Themes** - Light and dark mode support

## Tech Stack

- **Frontend:** D3.js, Chart.js, Plotly
- **Backend:** Python, Flask
- **Data Processing:** Pandas, NumPy
- **Database:** SQLite/PostgreSQL

## Installation

`ash

pip install -r requirements.txt

`

## Usage

1. Run the application
2. Upload your dataset (CSV/JSON)
3. Select visualization type
4. Customize appearance and export
